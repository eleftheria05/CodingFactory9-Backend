# CodingFactory9-Backend

REST API για διαχείριση χρηστών, authentication και role-based authorization, υλοποιημένο με Node.js, Express, TypeScript και MongoDB.

## Features

- User Management
- JWT Authentication
- Google OAuth Authentication
- Role-Based Authorization (RBAC)
- MongoDB & Mongoose
- Request Validation με Zod
- Swagger API Documentation
- Integration Testing με Jest
- TypeScript Support
